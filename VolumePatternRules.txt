# 巻数・話数判定の「追加」ルール（UserConfig ではなくこのテキストだけで編集）
# 個人用に VolumePatternRules.local.txt を置くと、このファイルのあとにマージされます（.gitignore 推奨）
# 本体の汎用パターン（第01巻・Vol.1 等）に加えて適用されます。空のまま（コメントのみ）でも挙動は従来どおりです。
#
# [ignore]   判定前に名前から除きたい語（部分一致・大小文字の区別なし）
#            ※部分一致のため、短い語（1〜2文字など）はタイトルや巻数表記まで削って壊しやすいです。
# [replace]  表記ゆれの置換「左側 => 右側」（最初の => で分割）
#            ※複数行あるときは、ファイル上から順にすべて適用されます（適用順で結果が変わります）
# [hint]     将来用（読み込みのみ。まだ判定ロジックには未接続）
# [regex]    上級者向け・将来用（読み込みのみ）
#
# 使用例（行頭の # を外すと有効。固有名はご自身の環境用に置き換え）:
# [ignore]
# 配布元のプレフィックス例
#
# [replace]
# Vol. => 第
# volume => 第
